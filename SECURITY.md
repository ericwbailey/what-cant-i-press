# Security Policy

## Supported versions

Only the latest release receives security fixes.

## Reporting a vulnerability

I ask that you do not open public Issue for security reports.

Please report a vulnerability privately via GitHub: **Security → Report a vulnerability** on [ericwbailey/what-cant-i-press](https://github.com/ericwbailey/what-cant-i-press/security/advisories/new).

Include reproduction steps, affected version, and platform (macOS or Windows). 

I take this seriously and will address the vulnerability as quickly as possible.
